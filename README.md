# 🎵 lyricExpert - 懂你想听，更懂他们唱了什么

> “你有没有好奇过，你最爱的歌手，一生中唱过多少次‘爱’，又提过多少次‘遗憾’？”

在这个快餐式听歌的时代，歌词往往被旋律掩盖。**lyricExpert** 是一款专为重度数字音乐爱好者、歌词考据党以及音乐行业从业者打造的**自动化中文歌词分析与洞察工具**。
它不仅仅能帮你把网易云音乐里的歌词批量扒下来，更是一位懂数据的“赛博乐评人”！

---

## 🌟 直击痛点：我们能为你做什么？

### 🔍 灵魂深处的「歌词检索」
*“我记得他唱过一句关于‘宇宙’的词，但想不起是哪首歌了……”*
**效果展示**：只需输入歌手名和关键词，瞬间翻遍该歌手的全部曲库。精确到句，甚至精确到时间戳以外的纯净文本。高级的去重和清洗逻辑，只留下最纯粹的表达，找回一段消失的记忆。

### ☁️ 艺术感拉满的「专属词云」
*“这位歌手的作词风格到底是怎样的？”*
**效果展示**：一键生成超清歌手专属词云图！内置强大的中文 NLP 分词引擎（支持停用词过滤、单字/英文剔除），扒开表象，用数据直观告诉你：TA 的世界里，除了“你我他”，还有怎样的日月星辰和喜怒哀乐。

### 📊 华山论剑：「跨歌手关键词排行」
*“乐坛里，谁才是真正的‘伤心情歌王’？谁最爱唱‘梦想’？”*
**效果展示**：输入特定词汇（比如“世界”），系统会自动遍历你本地已经抓取的所有歌手，计算这个词在他们各自作品中出现的频次和占比，自动生成同台竞技排行榜。数据流的浪漫，一目了然！

### 🤖 赛博乐评人：「AI 深度情绪解读」
*“字我都认识，但我想看看更深层的文学解读。”*
**效果展示**：接入最新强大的 AI 模型（如 DeepSeek 等），基于歌手的高频词汇与代表作数据，自动生成千字级别的深度乐评。解读 TA 歌词里的“感伤”、“叛逆”或“治愈”底色，给你独一份的听歌共鸣。

### 📦 粉丝福音：「一键同步关注列表」
*“我喜欢的歌手太多了，一个个搜太麻烦！”*
**效果展示**：一键绑定账号 Cookie，自动拉取你的网易云“关注艺人”列表，支持批量挂机下载曲库与歌词。睡一觉醒来，你的专属绝版歌词数据库就建立完成了！

## 🚀 极简启动：让每一次听歌都有迹可循

告别繁琐的配置，立刻拥有你自己的私人音乐分析工作站：

### 1. 环境准备
确保你已经安装了 Python 3.9+，然后安装所需依赖：
```bash
pip install -r requirements.txt
```

### 2. 启动服务
一键启动华丽的可视化 Web 工作台：
```bash
python app.py
```
然后在浏览器中打开 [http://127.0.0.1:5000](http://127.0.0.1:5000) 开始你的探索之旅！

### 3. 解锁 AI 隐藏玩法（可选）
如果想体验惊艳的 **AI 情绪分析**功能，只需在终端中配置对应的环境变量（默认兼容 OpenRouter 规范与 DeepSeek 等大模型）：
```powershell
$env:OPENROUTER_API_KEY="你的_API_KEY_填这里"
python app.py
```
